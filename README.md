## How to use?

1. Download or clone this repo.

```js
git clone  https://github.com/jones345/react-native-farm-health.git

```

2. Install dependencies.

```js
npm install
// or
yarn install

//or 

```

3. Run project on iOS / Android.

```js
 npm run ios // npm run android
 // or
 yarn ios // yarn android
```
